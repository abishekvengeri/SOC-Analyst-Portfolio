# Threat Intelligence – SOC Analyst Portfolio

This folder contains threat intelligence (TI) work including IOC enrichment, threat reports, TI workflows, and automation scripts.

## Structure
- **ioc-datasets/** — Sample IOCs (IPs, URLs, domains, hashes)
- **scripts/** — Python-based IOC enrichment tools
- **reports/** — Threat intelligence reports and analysis summaries
- **playbooks/** — TI investigation playbooks and workflows

## Objectives
- Analyze Indicators of Compromise (IOCs)
- Enrich indicators using external threat intelligence platforms
- Generate automated enrichment reports
- Build TI workflows suitable for SOC Tier 1 & Tier 2 operations

This folder demonstrates the ability to integrate TI sources into SOC workflows and create actionable insights.
